# Support

## Scope

This project supports:

- installer behavior (`init`, `doctor`, `version`)
- template integrity and command/script contract consistency
- documentation and onboarding correctness

## Out of scope

- custom third-party infrastructure debugging
- project-specific app bugs unrelated to installer/template

## Getting help

When opening a support request, include:

- OS and Node version
- command executed
- output/error logs
- whether `--force` and `--with-browser-skills` were used
